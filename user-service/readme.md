## USER SERVICE

- The User Service manages user registration, authentication, and user preferences for news based on country or categories .
- It ensures secure communication and implements authentication mechanisms,using JWT  to safeguard user data.
- This service implement caching strategies for news contents based on user preference, improving response times.
- The User Service plays a crucial role in facilitating communication between the Content Delivery Service and the News Aggregator Service to provide a personalized news experience for each user.
